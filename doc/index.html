概要¶

アプリ名（仮名）¶

グッシェア！機器登録初期設定説明書（スマートフォン用）

バージョン¶

Ver. 1.0

依頼元¶

MELCO リ本/電住家電/住ソ推/住ソ推・営企（担当：伊藤 慎一 様）

アプリ概要説明¶

グッシェア！の使用に必要な機器登録（エアコン／環境センサー／ファン用スマートスイッチ）について、初期設定方法を簡易的に説明するスマートフォン用マニュアルです。説明に使用している画面はMyMUの機器登録説明書から支給されています。

おもな機能¶

  • 画面をスワイプすることでページ送りが可能
  • 説明対象の箇所に吹き出しを設定
  • 補足説明はモーダルウインドウを利用
  • 縦長の操作説明画面で対象箇所が隠れてしまう場合、自動スクロールで説明箇所を表示

動作環境¶

対応デバイス¶

  • 一般的なPC／スマートフォン／タブレット
    iPhone SE（横幅750px）の画面を基準レイアウトとしています。

対応OS（iOS／Android／Windows）¶

  • 各デバイスの最新OS（2025年8月時点）
  • macOSは未検証

実機検証した端末名¶

  • Apple iPhone 13（iOS 18.6.2/1170×2532）
  • Google Pixel 8a（Android 10/1080×2400）

エミューレーター¶

  • LG it LGV36
  • Samsung Galaxy A21 SC-42A
  • Sony Xperia XZ1 Compact SO-02K

ファイル構成¶

GoodShareSettingsApp
│
├─assets
│  ├─css（スタイルシート）
│  ├─img（画面ファイル）
│  │  └─favicon
│  └─js（JavaScript）
├─partials（HTMLテンプレート）
├─rac（エアコン用説明 HTMLファイル）
├─sensor（環境センサー用説明 HTMLファイル）
└─switch（スマートスイッチ用説明 HTMLファイル）

JSONプロパティ名と値¶

プロパティ名 内容 値、および値の例
jsonName JSON名 GoodShareSettingsDemoApp
version バージョン 1.0.0
slides.index スライドのインデックス番号 0
slides.imagePath スライドに表示する画像ファイル名のパス ../assets/img/05-01.png
slides.alt スライドに表示する画像のimg要素のalt属性 STEP-01
slides.tooltipTarget ツールチップの対象（div要素で囲んだエリア） targetArea01
slides.tooltipDescription ツールチップに表示するテキスト ☰メニューをタップする
slides.footerContent 補足説明に表示するインナーHTMLのID名 dummy00
slides.modal 補足説明に表示されているテキストをクリックした時に表示されるモーダルのID名 modalRac001
